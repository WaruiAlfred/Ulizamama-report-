Double response trend — same question answered by both agent & instant
Overlap improvement: Double responses — where both an agent and UlizaMama replied to the same question (sometimes with identical content) — fell 66% from December to February. This indicates improving routing logic or handoff controls, but the remaining ~2,620/month still represents redundant messages worth investigating for user experience impact.

Key findings summary
1
Instant Response is scaling
UlizaMama went from handling 36.8% to 48.5% of all outgoing messages. However instant responses are generated with up to 18-part responses for a single question response.
2
Agent load significantly reduced
Agent-originated responses dropped from 39,151 to 11,726 — a 70% drop. However a majority of agent responses are 3–4 part messages with extremes of up to 22 parts.
3
Double responses improving
Concurrent responses from both channels are trending down 7,616 → 2,620, suggesting better routing controls. Although these responses go out up-to-8-part double messages from both agent and instant responses. Some of these double responses are sent out consecutively without even being escalated by the mums.
4
3 and 4-part messages dominate
Across both instant and agent responses, 3-part replies are the majority format.
